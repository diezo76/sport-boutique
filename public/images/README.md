# Images produits FollowMee

Les images de shooting sont disponibles sur Google Drive :
https://drive.google.com/drive/folders/1kUG3WmAeB34QZkbuAtJ09bfxMPvOkDpT

## Structure des dossiers Drive
- **1. T-shirts** – Photos de t-shirts
- **2. Sweet** – Photos de sweats
- **3. Casquettes** – Photos de casquettes
- **3. Groupe** – Photos de groupe
- **4. Equipe** – Photos équipe

## Comment ajouter les images

1. **Télécharger** les images depuis le Drive (clic droit sur le dossier → Télécharger)
2. **Placer** les fichiers dans ce dossier ou dans les sous-dossiers :
   - `t-shirts/`
   - `sweet/`
   - `casquettes/`
   - `groupe/`
   - `equipe/`

3. Les images seront accessibles via `/images/nom-du-fichier.jpg`

## Alternative : WordPress/WooCommerce

Pour une intégration automatique, uploadez les images dans votre boutique WooCommerce (followmee-officiel.com). Elles seront alors récupérées via l’API GraphQL avec les produits.
